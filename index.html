TeleportTo

Teleports the actor to the specified position

Parameters

  • positon - The vec3* coordinates of the position to teleport to

Examples

TeleportTo({ 10, 20, 30 });

Teleports the actor to the position (10, 20, 30)

  

DisplayClientMessage

Displays a client sided chat message to the actor

Parameters

  • message - The TextHolder* (or std::string*) that contains the message to be sent

Examples

DisplayClientMessage("[System] Hello world!");

Sends the message "[System] Hello world!" to the actor

  

Swing

Both starts the swinging animation and tells the server you are swinging

Parameters

  • No parameters.

Examples

Swing();

Causes the actor to server sidedly swing (LP)

getLocalPlayer

Returns an Actor* of the games localplayer if there is one else nullptr

Parameters

  • No parameters.

Examples

Actor* localPlayer = getLocalPlayer();

Gets a pointer to the local player

  

getBlockSource

Returns the BlockSource* pointer

Parameters

  • No parameters.

Examples

BlockSource* source = getBlockSource();

Gets a pointer to BlockSource
